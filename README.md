# 📸 Object Detection Project

## 📌 Description
This project showcases a simple object detection application that can identify common objects such as a person and a cellphone. The system uses a pre-trained model to detect and label objects in real-time or from images.

## 🎯 Objectives
- To detect objects using computer vision
- To understand how object detection models work
- To analyze factors affecting detection accuracy

## 🔍 Detected Objects
- Person
- Cellphone
- Cup
- Toothbrush
- Spoon

## ⚙️ Factors Affecting Accuracy
Detection accuracy may vary depending on:
- Lighting conditions
- Image quality or resolution
- Object size and distance

## 🛠️ Technologies Used
- Python
- OpenCV
- YOLO / Object Detection Model
